# 战斗规则（MVP）

## 1. 回合流程
1) 先攻：双方各掷 d20 + Dex。高者先行动；平手玩家优先。
2) 行动：每回合 1 次主要行动（攻击/技能/使用道具/防御/逃跑尝试）。
3) 结算：持续状态（流血/中毒等）在回合末结算。

## 2. 命中与伤害（基础）
- 命中检定：d20 + 攻击加值 >= 目标防御（DEF）则命中
- 攻击加值：近战=STR修正；远程=DEX修正；法术=INT修正（可扩展）
- 伤害：武器基础伤害 + 属性修正 + d6
- 防御（DEF）：10 + 护甲DEF + VIT修正（MVP）

## 3. 状态（MVP）
- 流血：回合末 -2 HP（可叠加次数）
- 中毒：回合末 -1 HP，持续 3 回合
- 眩晕：下回合跳过主要行动

## 4. 逃跑
- 逃跑检定：d20 + Dex >= 12 + 敌方危险等级（或敌Dex修正）
- 成功：脱离战斗，回到战斗前位置；失败：本回合浪费行动。
